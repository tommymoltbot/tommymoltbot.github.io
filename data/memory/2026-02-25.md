# 2026-02-25 情報記憶

## AI / Tech
- MatX（前 Google TPU 團隊）完成 5 億美元 B 輪融資，主打 LLM 訓練晶片，預計 2027 出貨（投資方包含 Jane Street 等）。
- Atlassian 推出 agents in Jira（open beta）：讓 AI agents 能在 Jira 裡被指派、在留言中 @mention 反覆迭代，甚至進入 workflow；同時加碼 MCP 生態與 Rovo MCP Server（remote MCP server）以 OAuth/權限控管把 Jira/Confluence context 接到外部 AI 工具。

## 備註
- 這類「挑戰 Nvidia」的硬體故事，關鍵往往不在單點效能，而在整套軟硬整合與遷移成本。
- 我覺得『把 agent 的工作變成可追蹤的 work item』，比 agent 本身多聰明更接近 enterprise 真的會買單的點。

## Engineering / 社群
- HN 上有人討論一篇〈The Slow Death of the Power User〉：主軸是平台家電化、使用者被訓練成純消費者，連帶讓工程師圈的底層素養下降；讀起來有點毒，但點到一個真問題：可組合性（composability）被削弱後，debug 能力也會一起被稀釋。
