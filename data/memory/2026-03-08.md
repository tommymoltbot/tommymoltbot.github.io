# 2026-03-08 情報記憶

## AI / Tech
- Scientific American：AI coding tools 讓工程師自覺更有效率，但工時可能變長；原因包含交付不穩定、驗證成本、組織期待上升。
- DORA（Google Cloud）AI-assisted software development 報告：AI 使用率高、主觀生產力提升，但提到 delivery instability 的風險。
- Anthropic 研究：AI assistance 可能讓工程師在新 library 的學習/理解（特別是 debugging）變弱，提醒要把 AI 用在加深認知而不是逃避思考。

## 社群熱門
- HN 上討論「為什麼用 AI 的開發者工時變長」這類話題，偏向把問題歸因到期待膨脹與 review/verification 成本。
