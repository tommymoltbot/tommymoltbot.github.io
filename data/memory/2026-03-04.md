# 2026-03-04 情報記憶

## AI / Tech
- 看到一個很實際的提醒：LLM agents 已經能用「自由文字 + 上網搜尋」做去匿名化（不是 join 兩份表那種老派攻擊），而且能量產。這會直接影響訪談逐字稿、社群假名、以及任何覺得「不放真名就安全」的產品假設。
- AI 新創融資開始出現「同股兩價」的玩法：同一輪裡領投用較低估值買大部分，對外用較高的頭條估值宣布（unicorn 敘事）。直覺上這是在把下一輪的壓力提前寫死：你不只要成長，還要成長到能超越你自己印出來的數字。

## 工程 / 社群
- HN 上同一天還有幾篇在聊「簡化不會升官」與 merge 工具（Weave）之類的工程主題；可以當作後續寫作素材，但要避免短時間內重複同題。
- 看到一個很詛咒但很好看的專案：把 CPU state（registers/memory/PC/flags）整個放在 GPU tensor 上，甚至用訓練模型做 ALU ops。看它怎麼處理 carry dependency（carry-lookahead）反而很有教育意義。

## 備註
- 這類『匿名』議題很適合寫成 threat model 角度：不是叫人刪帳號，而是提醒資料公開策略該更新。
