去年九月宣布的時候，很多人覺得這是「強強聯手」。Nvidia 要投資 OpenAI $100B，兩個 AI 時代最會賺錢的公司要綁在一起了。結果呢？WSJ 報導說這案子基本涼了，Jensen Huang 親口說金額「絕對不會像 $100B」，Nvidia 內部還有人對這交易表達疑慮。

從熱烈到擱置，才半年。

我第一反應是：「又來了。」科技圈這種宣示性交易，講得很大聲，最後不了了之的，也不是第一次了。但仔細想，這次有點不一樣。Nvidia 和 OpenAI 都不是小公司，$100B 也不是隨便說說的數字。能讓這案子卡住，背後應該有些更深的計算。

## 表面說法 vs 實際盤算

Nvidia 說還會參與當前融資輪，只是金額不會那麼大。聽起來像是「商業考量調整」，但我覺得沒這麼簡單。

想想 Nvidia 現在的位置：它賣晶片給所有 AI 公司，包括 OpenAI 的競爭對手。如果突然變成 OpenAI 的大股東，其他客戶會怎麼想？Google、Meta、Anthropic 這些公司會繼續放心跟 Nvidia 買嗎？還是開始擔心自己的資料、需求、計畫會不會被「無意中」透給 OpenAI？

從這個角度看，$100B 投資不是「強強聯手」，是「自己給自己挖坑」。

OpenAI 這邊呢？它需要的是錢和算力。Nvidia 可以給算力（賣 GPU），但如果要給到 $100B 的錢，代價是什麼？股權？決策權？OpenAI 的管理層願意讓 Nvidia 進來嗎？尤其是經過去年 Sam Altman 被踢又回來那齣，我很難想像 OpenAI 的董事會現在願意讓另一個巨頭加入這個局。

所以表面上是「交易擱置」，實際上可能是雙方都算清楚了：這筆交易對自己不划算。

## Nvidia 的兩難

Nvidia 現在的處境很微妙。它是 AI 時代最大的贏家，但這個贏法有點尷尬。

賣鏟子給挖金礦的人，這策略在淘金熱時期很聰明。問題是，如果你同時又投資了某個特定金礦，其他礦工還會跟你買鏟子嗎？

我之前看過一些分析說，Nvidia 想從「硬體供應商」變成「AI 生態的核心」。聽起來很合理，但實際操作很難。因為一旦你開始選邊站，你就失去了中立性。而中立性，是 Nvidia 現在能同時賺所有 AI 公司錢的基礎。

當然也有可能 Nvidia 根本不在乎這個。它可能覺得未來就是幾家大公司壟斷 AI，與其保持中立，不如早點選好隊伍。但如果真是這樣，為什麼現在又退縮了？

我猜可能是內部有人算了算，發現現階段還是繼續賣晶片比較安全。畢竟 AI 這局還沒定，太早站隊風險太高。

## 這對工程師意味著什麼

說實話，這件事對我們這些第一線寫 code 的人，短期內應該沒什麼影響。

Nvidia 的 GPU 該多貴還是多貴，OpenAI 的 API 該怎麼用還是怎麼用。這筆交易成不成，不會改變我明天能不能跑得動模型，或是能不能 afford 得起 API call。

但長期來看，這反映了一個趨勢：AI 這個領域，大公司之間的關係越來越複雜。今天是合作夥伴，明天可能是競爭對手，後天可能又想併購。這種不確定性，對整個生態是有影響的。

我比較在意的是，當這些大公司在盤算各自的利益時,他們會不會順便把技術發展的方向也帶歪了？比如說，為了商業考量而選擇閉源、為了壟斷而故意設定技術壁壘。這些對技術本身的進步，其實是有害的。

## 見怪不怪了

老實說，這種「宣布大計畫然後不了了之」的劇本,我已經看過太多次了。

2010 年代，多少家公司宣布要做「革命性」的東西，最後都沒下文了。AI 時代這幾年，hype cycle 轉得更快，宣示性的合作案更多。大部分都是為了股價、為了 PR、為了讓投資人覺得「我們在做事」。

Nvidia-OpenAI 這個案子,從宣布到擱置才半年,算是收得還算快的。至少比那些拖個兩三年最後悄悄不提的好。

我的判斷是：這案子大概不會成了。雙方還會保持合作關係（OpenAI 繼續買 Nvidia 的 GPU，Nvidia 可能象徵性投一點錢），但 $100B 那個規模的交易？算了吧。

這事最後會怎樣？我猜過幾個月就沒人記得了。AI 圈的新聞太多，熱度過得很快。等下一個「突破性合作」宣布的時候，大家又會興奮一輪。然後呢？可能又是一樣的結局。

見怪不怪了。
