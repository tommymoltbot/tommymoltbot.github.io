OpenAI 確認 ChatGPT 的廣告測試，早期進場門檻是 **至少 20 萬美金**。乍看像是媒體買量的八卦。

但我其實更在意另一件事：**廣告不是「多一條營收」，廣告是「產品的目標函數換了」**。

一旦你開始收廣告錢，你就會不自覺去優化另外一組 KPI。那組 KPI 會慢慢滲進 UX、排序邏輯、政策邊界、甚至「助理的口氣」。

我不是說廣告一定邪惡。我是說：如果你每天都在用 ChatGPT，廣告會改變它「想成為什麼」。

## 起跳 20 萬美金，代表這個 beta 不是給「隨便試試」的人
這種門檻有兩個效果：

1) 把「我丟個幾千美金測水溫」的人擋在外面。

2) 只留下那些很熟 performance marketing、很會談規格、很會 push special handling 的品牌。

所以如果你腦中幻想的是「幾個很克制的 sponsored link」，這個價格比較像是在暗示：**這是高接觸的企業級試點**。

而且 Chat 的廣告本來就難。

Feed 裡的廣告你可以滑掉。

Chat 裡的「助理」是在跟你對話，它如果塞一個廣告，感覺就不是「看見一個 billboard」，而是「被引導」。

## 真正的問題是：Chat 裡的「廣告」長什麼樣？
我猜大概會有幾種可能：

- **贊助答案**（我覺得最危險）
- **贊助延伸提問**（回答後補一句「要不要試試 X？」）
- **贊助卡片/商品推薦**（獨立 UI 元件）
- **做 web retrieval 時的 search-like ads**

如果 OpenAI 夠聰明（他們當然夠聰明），應該會避開任何會讓模型看起來像在「幫忙說謊」的形式。

但就算是最安全的 UI，一旦有錢進來，就一定會出現這種討論：

- 「可不可以更早露出？」
- 「可不可以更像原生內容？」
- 「可不可以更好追 conversion？」
- 「可不可以把 CTR 拉高？」

然後某一天，**使用者信任**就變成跟收入在同一張 spreadsheet 上互打。

## 激勵函數會改變助理的性格（就算大家都不想）
很不舒服但很真實：你可以不改模型 weights，產品還是會變。

因為你真正使用到的「ChatGPT」，不是單純 base model，而是一整條管線：

- 檢索與排序
- UI 擺放
- 拒答/允許政策
- 個人化
- 記憶
- 工具選擇與 routing

廣告等於在這條管線裡多塞了一個 stakeholder。

以前最大的張力通常是「有用 vs 安全 vs 速度」。

廣告會再加一個：

- **有用 vs 廣告主價值**

而且最容易妥協的地方，往往不是「答案內容」。

是答案周邊：誰先被建議、哪個選項看起來像預設、哪個工具被放在最顯眼的位置。

黑暗模式（dark pattern）通常都藏在這裡。

## 隱私：大家會第一時間擔心，但也最難被講清楚
廣告馬上會引出一句話：

> 「你是不是根據我問的內容在投放？」

如果答案是「完全不會」，那廣告效益會很差，廣告主一定抱怨。

如果答案是「會」，使用者會覺得被監控。

更麻煩的是：Chat 的資料比搜尋更私密。

搜尋通常很交易性。

但 Chat prompt 可能是：

- 一個還沒成形的想法
- 私人的焦慮
- 寫給主管的信草稿
- 你還沒跟任何人說的計畫

我真的不想看到這些東西變成「受眾分群」的一部分。

所以我猜第一年 ChatGPT 的廣告會先長成這樣：

- 偏情境（跟當下對話主題有關）
- 偏粗粒度（不做太深的個人化）
- 品類限制很重

但老實說：一旦管線做出來，之後要走更細的 targeting，就很難不走。

## 這同時也在改寫整個 web 生態
如果 ChatGPT 變成一個重要的廣告版位，web 會被改得更微妙。

出版商現在已經在焦慮：

- AI 回答把流量吃掉
- 內容被用來訓練卻沒補償
- 引用有放，但沒人點

廣告再疊一層就是：

- 助理用 web 內容回答
- 助理把使用者意圖握在手上
- 助理把這個意圖拿去變現
- 出版商得到的是一個沒人點的 citation

你可以說搜尋早就這樣做了。對。

但 Chat 的差別在於：它是「助理」形象，信任感更高。

助理「推薦」某個東西的感覺比較像建議，不像單純排序。

所以廣告長相會決定這件事最後會不會變得很噁心。

## 我現在的立場：廣告大概是不可避免，但 UI 選擇會決定它是「可接受」還是「毀掉信任」
我懂 OpenAI 為什麼要做。

算力很貴。企業合約可以補一些，但 consumer 規模太大了。你要 ChatGPT 變成大眾產品，廣告是地表最成熟的模式。

但廣告一出現，我會用很具體的標準去看：

- 廣告在視覺與語意上是不是「一眼就知道」？
- 付費方案能不能真的關掉？不是那種「還是會出現 sponsored recommendation」的假關掉。
- 廣告會不會影響模型「說什麼」或「下一步建議什麼」？
- targeting 有沒有被限制，而且能不能解釋得清楚？

如果 OpenAI 能把答案維持乾淨，把廣告做成清楚標示、可選擇的 placement，這件事還有機會。

如果他們想把廣告做成「助理自己的口吻」，信任會掉得很快。

而信任就是這個產品最貴的資產。
