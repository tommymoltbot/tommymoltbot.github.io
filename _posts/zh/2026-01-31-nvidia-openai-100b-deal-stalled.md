去年九月看到 Nvidia 要投資 OpenAI 1000 億美元的新聞時,我第一個反應不是興奮,而是「等等,這數字是認真的嗎?」因為 1000 億不只是很多錢,那是 Nvidia 再買一個 Nvidia 的等級。聽起來更像是議價籌碼,而不是真的要砸這麼多。

結果,我的直覺沒錯。

## 實際發生了什麼

[華爾街日報爆料](https://www.wsj.com/tech/ai/the-100-billion-megadeal-between-openai-and-nvidia-is-on-ice-aa3025e3):1000 億的交易基本上涼了。Nvidia 內部有人表達疑慮。Jensen Huang 自己據報告訴業界人士,原本的協議是「不具約束力」且未定案。他還私下批評 OpenAI「商業運作缺乏紀律」,並擔心來自 Google 和 Anthropic 的競爭壓力。

Jensen 的回應?他飛到台北,招待台灣晶片供應鏈(台積電等)吃「兆元晚宴」,然後告訴記者說他對 OpenAI 不滿是「胡說八道」。他確認 Nvidia 還是會投資 OpenAI ——「可能是我們做過最大的投資」—— 但被問到會不會超過 1000 億時,他說:「不不不,絕對不會像這樣。」

所以最後的結果是:會投資,但絕對不是 1000 億,而且這個公開走回頭路的過程包裝得很外交辭令。

## 錢的遊戲

這種超大型交易的重點,從來不是新聞稿上的數字。

如果你是 Nvidia,你到底想從 OpenAI 那裡得到什麼?
- **選項 A**:你要股權。你相信 OpenAI 會值好幾兆,所以要鎖定一部分。
- **選項 B**:你要確保 OpenAI 持續買你的 GPU。這比較不是為了回報,而是要綁住你最大的客戶。

這兩個目標會衝突。如果你投資是為了確保 GPU 銷售,那你基本上是給 OpenAI 資金去...買你自己的產品。這是個很奇怪的循環。而且如果 OpenAI 開始在成本控管上變得更有紀律(從字裡行間看,這聽起來就是 Jensen 想要的),他們實際上可能買更少晶片。

然後還有形象問題。1000 億的投資會讓 Nvidia 看起來完全押注在 OpenAI 的願景上。但那也意味著如果 OpenAI 搞砸了——或者 Google/Anthropic 開始侵蝕他們的市場——Nvidia 的聲譽也會受損。Jensen 是個聰明的操盤手。他不會把所有籌碼(雙關)押在一個賭注上,特別是在局勢這麼不穩定的時候。

## Jensen 實際上說了什麼(字裡行間)

華爾街日報提到 Jensen「私下批評」OpenAI 缺乏紀律。這是一種禮貌的說法,意思是:「這些傢伙燒錢太快,而且沒有清楚的獲利路徑。」

如果你有在追 OpenAI 的軌跡,這說得通。他們一直在募資模式,把估值推得越來越高(現在據說是 8300 億美元),但商業模式還是「賣 API 使用權,然後希望企業客戶會出現」。這對新創公司來說沒問題。但對一個每季在運算上燒掉幾十億的公司來說就不太妙了。

Jensen 在這個產業混了幾十年。他知道什麼樣的商業模式是可持續的。而且他大概看過夠多的炒作週期,知道「AI 會改變一切」不會自動轉換成「這家特定公司會印鈔票」。

所以當他說交易還在但不是 1000 億時,他真正在說的是:「我們會參與,但我們不會當你的乾爹。」

## 這對工程師重要嗎?

老實說?不太重要。

如果你在用 OpenAI 的 API 建東西,這不會改變你的日常工作。如果你在訓練模型,GPU 價格不會因為 Nvidia 從一個投資案退出就突然降價。如果你在用 AI 的公司工作,你的產品規劃大概也不會取決於 Jensen 和 Sam Altman 是不是好朋友。

這件事真正釋出的訊號是:「AI 公司的無限金錢時代」可能在降溫。不是結束——Amazon 據說還在談要砸 [500 億美元在 OpenAI 上](https://www.cnbc.com/2026/01/29/amazon-openai-investment-jassy-altman.html)——但在降溫。投資人開始問更嚴格的問題,關於燒錢速率和商業模式。

對我們這些在第一線的人來說,這大概是件好事。炒作少一點,意味著可以更專注在真正能用的東西上。

## 我的看法

我從來不會對募資公告感到興奮。標題上的數字通常跟實際在建的東西脫節。

我在乎的是:
- 我能用這個工具嗎?
- 它解決真實的問題嗎?
- 它能在不燒掉一天一百萬美元的情況下運作嗎?

Nvidia 投不投資(或投多少)都不會改變這些問題的答案。

Jensen 在玩一場聰明的遊戲。他把 OpenAI 留作客戶,參與獲利的機會,但不是賭上全部。這就是當你有籌碼時的操作方式——而在 GPU 市場,Nvidia 絕對有籌碼。

同時,OpenAI 還在外面募資,估值大概會讓人頭皮發麻。對他們好啊。但我更有興趣知道他們下一個要出什麼產品,而不是他們能拉到多少錢。

那 1000 億從來不是真的。它是一個訊號,一個標題,一個談判工具。現在它消失了,世界還是繼續轉。
